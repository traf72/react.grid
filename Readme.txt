Проект представляет из себя надстройку над библиотекой https://griddlegriddle.github.io/v0-docs/
Добавлен функционал фильтрации, множественного выбора записей, экспорта в Excel и т.п.

Для запуска примера необходимо:

	1. Из корня проекта выполнить команду (npm должен быть установлен и добавлен в PATH):
		npm install
		
	2. Из корня проекта выполнить команду:
		node_modules\.bin\webpack
		
	3. Запустить файл index.html

Для запуска unit-тестов необходимо из корня проекта выполнить команду:
	npm test
	
Весь код, кроме папки js\src\libs\ написан лично.

Это кусок из реального проекта. Не всегда код соответствует React Best Practices, например, в текущих версиях React вместо метода componentWillReceiveProps лучше использовать getDerivedStateFromProps, а для решения о том, должен ли компонент обновляться, нужно использовать метод shouldComponentUpdate. По большей части это из-за того, что данный компонент писался на заре знакомства с React, а также в некоторых случаях приходилось подстраивать код под уже существующую библиотеку, чтобы не лезть в её исходники.
